## 3-Minute Setup

### 1. Supabase Configuration

Go to your Supabase Dashboard:

```
1. Database → Tables → braider_profiles
   → Realtime tab → Toggle ON
   → Select: INSERT, UPDATE, DELETE

2. Storage → Create bucket "braider-avatars" → Public
3. Storage → Create bucket "portfolio-images" → Public
4. For each bucket: Policies → New Policy → For public access → Save
```

### 2. Start the App

```bash
npm run dev
```

App runs on `http://localhost:3000`

### 3. Test

```
1. Sign up as braider
2. Check homepage - braider appears in 2 seconds
3. Add portfolio item with 6+ images
4. Verify images upload
```
