## How to Test

### Test Carousel
1. Go to landing page (`/`)
2. Scroll to "Featured Braiders"
3. Watch it auto-rotate
4. Click arrows to navigate

### Test Avatar Upload
1. Login as braider
2. Go to `/braider/dashboard`
3. Click upload button on avatar
4. Select image file
5. Verify it saves and displays

### Test Booking
1. Login as customer
2. Go to `/booking`
3. Select braider, service, date, time
4. Click "Confirm Booking"
5. Go to `/dashboard` → "My Bookings"
6. Verify booking shows with status

### Test Verification
1. Login as braider
2. Go to `/braider/verify`
3. Upload file for Step 1
4. Watch it auto-advance
5. Repeat for all steps

### Test Responsive
1. Open any page
2. Resize browser to mobile width (375px)
3. Verify single column layout
4. Resize to tablet (768px)
5. Verify 2-column layout
6. Resize to desktop (1440px)
7. Verify 3-4 column layout
